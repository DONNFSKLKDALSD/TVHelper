# 影视助手
## 痛点
- 使用在线配置，不方便自定义
- 在线配置缓存本地，担心更新不及时

## 功能（详见`config/sample.json`）
- 极高的自定义程度
- 多源整合、处理（在线缝合）
  - 去重
  - 失效切换备用源
  - 多jar
- 点播源黑名单，指定名称的点播源不再展示
- 豆瓣主页
- ...
